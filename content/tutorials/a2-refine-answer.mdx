## 이 단계에서 배울 것

첫 답변이 완벽할 필요는 없습니다. **덧붙여 요청**하면서 원하는 결과에 가까워집니다.

## 해보기

이전 단계의 답변에 이어서, 아래처럼 방향을 더해 보세요.

```
좀 더 친근한 말투로 바꿔 주고, 마지막에 격려 한 문장을 추가해 줘.
```

## 정리

- 고칠 점을 **콕 집어** 말할수록 좋습니다. ("더 좋게" ❌ → "더 짧게, 두 문장으로" ✅)
- 마음에 드는 부분은 "이 부분은 그대로 두고"라고 알려주면 유지됩니다.
