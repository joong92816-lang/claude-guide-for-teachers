## 이 단계에서 배울 것

AI에게 일을 잘 시키려면 **무엇을 · 누구에게 · 어떤 형식으로**를 함께 알려주는 것이 핵심입니다.

## 해보기

아래 질문을 그대로 **질문하기** 창에 넣어 보세요.

```
초등학교 5학년 학생들에게 '분수의 덧셈'을 설명하는
3줄짜리 쉬운 안내 문구를 만들어 줘.
```

## 정리

- **무엇을**: 분수의 덧셈 안내 문구
- **누구에게**: 초등 5학년
- **형식**: 3줄

세 가지가 들어가면 답변의 품질이 크게 좋아집니다. 다음 단계에서는 답변을 다듬는 법을 배웁니다.
