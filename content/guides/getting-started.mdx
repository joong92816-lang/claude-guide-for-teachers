## ① 한 줄 요약

클로드 코드는 컴퓨터에게 우리말로 일을 시키는 도구입니다. 설치 후 "무엇을 해줘"라고 말하면 됩니다.

## ② 이럴 때 써요

- 반복되는 문서 작업(안내문, 표, 목록)을 빠르게 초안잡고 싶을 때
- 자료를 요약하거나 정리하고 싶을 때
- 아이디어가 필요할 때 (활동 설계, 발문 만들기 등)

## ③ 준비물

- 인터넷이 되는 컴퓨터
- 이 서비스 화면 (별도 설치 없이 브라우저에서 바로 사용)

## ④ 따라 하기

1. 상단의 **질문하기** 버튼을 누릅니다.
2. 입력창에 하고 싶은 일을 우리말로 적습니다.
3. **보내기**를 누르고 답변을 기다립니다.
4. 답변이 마음에 안 들면, 원하는 방향을 덧붙여 다시 질문합니다.

## ⑤ 예시 프롬프트

```
우리 반 학부모 총회 안내문을 정중한 말투로 초안 잡아 줘.
날짜는 다음 주 금요일 오후 7시, 장소는 3학년 2반 교실이야.
```

## ⑥ 주의할 점

- 학생의 실명·연락처 같은 개인정보는 입력하지 마세요. "학생1"처럼 익명으로 적으면 됩니다.
- AI의 답변은 초안입니다. 보내기 전에 꼭 한 번 읽고 다듬어 주세요.
